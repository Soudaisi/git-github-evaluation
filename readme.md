Qu’est-ce que Git et GitHub ?

Git est un outil de gestion de versions qui permet de suivre l’évolution d’un projet informatique. Il enregistre les modifications apportées aux fichiers sous forme de commits, ce qui permet de revenir à une version précédente en cas d’erreur. Git fonctionne en local sur la machine du développeur et facilite le travail organisé, individuel ou en équipe.

GitHub est une plateforme en ligne qui permet d’héberger des dépôts Git à distance. Elle sert à sauvegarder le code, à le partager avec d’autres développeurs et à collaborer sur un même projet. GitHub offre une interface web ainsi que des fonctionnalités collaboratives comme le suivi des modifications, la gestion des projets et le travail en équipe.

En résumé, Git est l’outil qui permet de gérer les versions d’un projet en local, tandis que GitHub est le service qui permet de stocker ce projet en ligne et de le partager.
